# 生日提醒系统

一个美观、功能齐全的生日提醒系统，支持公历和农历生日提醒。

## 功能特性

### 核心功能
- ✅ 记录亲朋好友的生日信息
- ✅ 支持公历和农历生日
- ✅ 精确到时分秒的生日时间设置
- ✅ 自动计算并显示未来3个月内即将过生日的人员
- ✅ 女性成年后不显示年龄（隐私保护）
- ✅ 男女/家人朋友分类管理

### 管理员功能
- ✅ 添加、编辑、删除成员信息
- ✅ 查看详细生命统计
  - 生命总天数
  - 精确年龄（年月天时分秒）
  - 距离下次生日的天数
- ✅ 选择任意日期查看某人的生命统计

### 访客功能
- ✅ 查看即将到来的生日
- ✅ 只读访问，保护隐私

### UI/UX
- ✅ 现代化渐变设计
- ✅ 响应式布局
- ✅ 平滑动画效果
- ✅ 直观的卡片式界面

## 技术栈

- **框架**: Next.js 16 + React 18 + TypeScript
- **样式**: Tailwind CSS + shadcn/ui
- **农历计算**: lunar-javascript
- **数据存储**: LocalStorage
- **图标**: Lucide React
- **通知**: Sonner

## 使用指南

### 首次使用

1. 访问应用，会自动跳转到登录页面
2. 选择角色登录：
   - **访客**: 仅查看生日列表
   - **管理员**: 可以添加、编辑、删除成员，查看详细统计

### 添加成员（管理员）

1. 登录后，点击右上角的"管理面板"按钮
2. 点击"添加成员"
3. 填写信息：
   - 姓名（必填）
   - 类别（家人/朋友）
   - 性别（影响年龄显示）
   - 生日日期（必填）
   - 生日时间（可选）
   - 是否为农历生日

### 查看生日列表

首页显示未来3个月内即将过生日的人员：
- 卡片颜色区分：家人（粉色），朋友（蓝色）
- 农历生日会显示"农历"标签
- 今天生日会显示特殊高亮
- 倒计时显示距离生日的天数

### 查看详细统计（管理员）

1. 进入管理面板
2. 切换到"详细统计"标签
3. 选择要查看的人员
4. 选择参考日期（默认为今天）
5. 查看：
   - 下一次生日及年龄
   - 生命总天数
   - 详细年龄（精确到秒）

## 隐私保护

- 成年女性不显示年龄
- 访客只有只读权限
- 数据保存在浏览器本地，不会上传到服务器
- 可配置人员可见范围（visibleTo 字段）

## 注意事项

1. 数据存储在浏览器的 LocalStorage 中，清除浏览器数据会丢失所有信息
2. 适合个人或小团队内部使用
3. 如需多用户同步功能，需要后端支持

## 本地部署方案
```bash
已完成容器化配置。以下是本地部署方案：
创建的文件
1. Dockerfile - 多阶段构建，优化镜像大小
2. docker-compose.yml - 简化部署管理
3. .dockerignore - 排除不需要的文件
4. 更新 next.config.ts - 启用 standalone 输出模式
部署方式
方式一：使用 Docker Compose（推荐）
# 构建并启动
docker-compose up -d
# 查看日志
docker-compose logs -f
# 停止服务
docker-compose down
# 重新构建
docker-compose up -d --build
访问 http://localhost:3000

方式二：使用本地 Node.js
npm install
npm run build
npm start

配置说明
- 端口: 默认 3000，可在 docker-compose.yml 中修改
- 重启策略: unless-stopped（除非手动停止，否则自动重启）
- 数据: 当前数据存储在浏览器 LocalStorage，无需服务端持久化，开了 /app/data 挂载没有用。
```
